 📢 Chat-App – A Modern Realtime Chat Application  


This is a full-stack real-time chat app built using modern technologies, inspired by a tutorial from [this YouTube video](https://youtu.be/ntKkVrQqBYY).

---

 🚀 Features
- 🌟 Tech stack: MERN + Socket.io + TailwindCSS + Zustand  
- 🔒 JWT-based authentication & authorization  
- ⚡ Real-time chat with Socket.io  
- 🟢 Online user status tracking  
- 🎨 Custom UI styling with Tailwind & Daisy UI  
- 🚀 Efficient state management using Zustand  
- 🐞 Robust error handling (client & server-side)  
- 🌍 Deployable for FREE!  

---

 🛠 Setup – Environment Variables (.env)
Before running the app, create a `.env` file and configure the following:  

```ini
MONGODB_URI=your_mongodb_connection_string
PORT=5001
JWT_SECRET=your_secret_key

CLOUDINARY_CLOUD_NAME=your_cloudinary_name
CLOUDINARY_API_KEY=your_cloudinary_api_key
CLOUDINARY_API_SECRET=your_cloudinary_api_secret

NODE_ENV=development
```
